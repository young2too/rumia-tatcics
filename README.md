# Lumia Tactics

이터널 리턴 캐릭터 이름을 활용한 팬 오토체스 웹 프로토타입입니다.

## 실행

현재 로컬 서버:

```text
http://127.0.0.1:8765
```

렌더러 서버:
```text
https://rumia-tatcics.onrender.com
```

정적 파일만으로 동작하므로 `index.html`을 브라우저로 직접 열어도 됩니다.

## 구현된 기능

- 캐릭터 상점, 새로고침, 구매
- 벤치와 보드
- 드래그 배치와 클릭 배치
- 라운드별 자동 전투
- 크레딧, 체력, 레벨업
- 2성/3성 합성
- 역할 기반 시너지(사실상 장식용. 밸런스 패치같은건 미래의 일)
